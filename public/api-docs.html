FrictionWell API Documentation

Base URL: https://frictionwell.drillsoft.com/api/v1

Public Endpoints

Get Subscription Plans

GET /subscription-plans

Returns all active subscription plans.

Get Latest App Version

GET /app-versions/latest

Returns the latest version of the application.

Protected Endpoints

These endpoints require authentication using a Bearer token.

Get User Profile

GET /user

Returns the authenticated user's profile.

Get User Subscription

GET /user/subscription

Returns the user's active subscription details.

Create Calculation

POST /calculations

Creates a new calculation for the authenticated user.

Get User Calculations

GET /calculations

Returns the user's calculation history.

Authentication

Protected endpoints require a Bearer token in the Authorization header:

Authorization: Bearer your-token-here
